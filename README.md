# Phoenix Vault DApp

A legacy recovery protocol for the Cardano Blockchain.

## Setup
1. Unzip the folder.
2. Run `npm install`.
3. Run `npm run dev`.
4. Open [http://localhost:3000](http://localhost:3000).

## Configuration
- API Key: Integrated in `.env.local`.
- Network: Preprod Testnet.
- Wallet: Works with Nami (ensure it's on Preprod).
